Software de cosecha, transformación y publicación de metadatos OAI-PMH

Requerimientos para la instalación.

La plataforma está testeada sobre Ubuntu 12.04LTS con el siguiente software de base:

  • Java Sun JDK preferentemente versión 1.7
  • Apache Tomcat (preferentemente versión 7.0)
  • MySQL version 5.5 o posterior.
  • Apache SolR (versión 3.6 o superior de la línea 3.x)
  • PHP 5.3 o superior
  • Apache webserver 2.2

Las instrucciones detalladas de instalación se encuentran en el wiki. La base de este desarrollo fue realizado en el marco de la Componente III del proyecto LA Referencia.
