
Activar el modo rewrite de apache2 y reiniciar apache
---------------------------------------------------------
sudo a2enmod rewrite
sudo /etc/init.d/apache2 force-reload

Es necesario dar permiso a Apache a lo siguiente
---------------------------------------------------------
sudo chown www-data:www-data /var/www/vufind/interface/compile
sudo chown www-data:www-data /var/www/vufind/interface/cache 
sudo chown www-data:www-data /var/www/vufind/images/covers/*

Copiar la carpeta completa de vufind a 
---------------------------------------------------------
/var/www/vufind

Ligar a Apache el vufind
----------------------------------------------------------
sudo ln -s /var/www/vufind/httpd-vufind.conf /etc/apache2/conf.d/vufind
sudo /etc/init.d/apache2 reload


Instalar Vufind
--------------------------------------------------------
cd /var/www/vufind
sudo /var/www/vufind/install

Agregar variables globales
--------------------------------------------------------
sudo sh -c 'echo export VUFIND_HOME=\"/var/www/vufind\"  >> /etc/profile'
source /etc/profile

Editar el archivo para apuntar a Solr y servicios Web de LA REFERENCIA
--------------------------------------------------------
sudo nano /var/www/vufind/web/conf/config.ini

Revisar la instalación
--------------------------------------------------------
http://localhost/vufind
